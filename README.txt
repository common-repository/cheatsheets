📦 Block for creating cheat sheets in WordPress (based on Gutenberg)

== Description ==

This plugin is just a separate block 💎, which helps users quickly create
cheatsheets in WordPress using Gutenberg and other default blocks!

-   [Demo](https://true-emotions.studio/cheatsheets/) 👀

-   For best effect, I advise you to use this block together with the standard block "columns"!

**Features 👓:**

-   Selectable colors for text, title and background.

-   Adjustable text-size and block border-radius.


**Take look at another our [plugin for table reservations](https://wordpress.org/plugins/table-reservations/) 🥂**

= How to start =

1.  Install [Gutenberg](https://wordpress.org/plugins/gutenberg/) (if you don't
    have)

2.  Create Columns block

3.  Create Cheatsheets block inside column
